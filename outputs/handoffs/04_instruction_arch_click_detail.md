## 设计任务

### 右侧面板当前结构

```
┌──────────────────────┐
│  对话 | 企业列表 | 自进化  │  ← 标签页表头
├──────────────────────┤
│                      │
│   标签页内容区域      │
│                      │
└──────────────────────┘
```

### 需要新增的点选详情面板

设计一个详情面板，满足：
- 点击网格时出现，不影响现有三个标签页的切换
- 与标签页在同一右侧面板内，但视觉上独立
- 可关闭，关闭后恢复标签页视图
- 展示网格属性（用地类型、面积亩、权属、乡镇、混合地类、地类占比 top3）

### 请设计 2-3 个方案

| 方案 | 思路 |
|------|------|
| A | 详情作为第四个标签页"详情"，点击网格时自动切到此标签 |
| B | 详情作为浮动卡片，覆盖在标签页内容上方（overlay），带关闭按钮 |
| C | 详情替换标签页内容区域，顶部保留关闭按钮和"返回标签页"链接 |

给出推荐方案及理由。
