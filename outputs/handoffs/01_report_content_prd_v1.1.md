**设计说明**：

1. **悬浮高亮选无动画**：直接 addTo/removeFrom map 而非 CSS transition，降低 Leaflet 层闪烁风险，且 80ms throttle 下动画无意义
2. **金字塔层级映射按视觉密度推导**：zoom 11 全县约 30 个 3200m 网格（城镇级），zoom 13 约 200 个 800m 网格（片区级），zoom 15 约 800 个 200m 网格，逐级递进
3. **聚合字段 `land_type_top3` 用 JSON 字符串而非独立表**：避免多表 JOIN 性能损耗，单格查询时反序列化即可
4. **降级策略保守**：聚合表不存在时 fallback 到 100m 原始数据 + 截断，确保功能可用但不因为预处理缺失而完全不可用
