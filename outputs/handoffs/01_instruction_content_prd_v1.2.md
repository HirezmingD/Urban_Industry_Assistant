## 修订 1：九宫格性能优化——预加载 + 透明切换

### 用户原始需求

> 跟地图一同加载该级别全部的网格，但网格边界和填充色都是透明的（无色），鼠标经过时再实时调整其颜色和透明度。

### 结构化翻译（PM 语言）

**功能名称**：九宫格预加载渲染

**现状问题**：每次 mousemove 发 HTTP 请求 → 等 WKB 解析 → 创建/销毁 GeoJSON Layer → 卡顿。

**新方案**：

| 步骤 | 行为 |
|------|------|
| 地图加载/zoom 切换 | 一次性加载当前 zoom 级别**全部网格**的 GeoJSON，渲染为透明 Layer（fillOpacity=0, strokeOpacity=0） |
| 鼠标移动 | 判断光标所在网格 → 仅修改该网格及周围 8 格的样式（fillOpacity/stroke/color），不创建不删除 Layer |
| 鼠标移出 | 恢复所有网格为透明 |

**分级策略**（网格数差异大，不能全量）：

| Zoom | 网格数 | 加载策略 |
|:--:|--:|------|
| 11 | ~270 | 全量预加载 |
| 12 | ~950 | 全量预加载 |
| 13 | ~3,600 | 全量预加载 |
| 14 | ~13,700 | 全量预加载 |
| 15 | ~53,800 | 视口内懒加载 + 视口外异步 |
| 16-18 | ~185,000 | API 按点查询（保持现有模式，太密无需全量） |

**性能指标**：
- mousemove 响应 < 16ms（单帧内完成，无网络请求）
- zoom 切换时首次加载 < 500ms
- 九宫格高亮切换无闪烁

**数据格式**：新增端点 `GET /api/map/grid_layer?zoom=11`，返回轻量 GeoJSON FeatureCollection（仅 geometry + grid_id，无 land_type/ownership 等属性）。

## 修订 2：点选弹窗——光标旁 sticky note

### 用户原始需求

> 用户点选网格，鼠标旁边弹出一个像即时贴一样的对话样式的标签，里面写上该地基本信息，交互应该是很精致的。

### 结构化翻译（PM 语言）

**功能名称**：点选即时贴（Sticky Note）

**触发条件**：
- 用户在地图上点击任意渔网单元
- 仅政府端生效

**视觉规格**：

```
┌─────────────────────┐
│ 📋 grid_120_45    × │  ← 标题栏（网格ID + 关闭按钮）
├─────────────────────┤
│ 用地类型  工业用地    │
│ 面积      15.0 亩    │  ← 属性行（左标签右值）
│ 权属      国有       │
│ 乡镇      桐君街道   │
│ 混合用地  含混合用地  │
├─────────────────────┤
│    [评估此地块 →]    │  ← 操作按钮
└─────────────────────┘
```

**位置约束**：
- 弹出位置：**光标右下方 12px 偏移处**（跟随点击位置，不固定）
- 如果光标靠近地图右边缘 → 向左弹出
- 如果光标靠近地图下边缘 → 向上弹出
- **不得遮挡地图中心区域和右侧面板**
- Z 序：高于所有地图图层，不随地图缩放/平移而移动（`L.popup` 或绝对定位 div）

**尺寸约束**：
- 最大宽度：280px
- 最大高度：320px（超出滚动）
- 字号：13px

**外观**：
- 白色/浅色背景，圆角 8px，轻微阴影
- 标题栏：浅灰底，左对齐网格 ID，右侧 × 关闭按钮
- 属性行：左标签灰色，右值深色
- "评估此地块"按钮：主题色，全宽

**关闭方式**：
- 点击 ×
- 点击地图空白处
- 再次点击同一网格（toggle）
- 点击另一网格（切换内容，不关闭再开）
- Esc 键

**过渡动画**：
- 弹出：从光标位置微缩放出现（scale 0.9→1.0, 150ms）
- 关闭：微缩放消失（scale 1.0→0.9, 100ms）
