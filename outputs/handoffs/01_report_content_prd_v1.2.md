**设计说明**：

1. **分级策略边界选在 zoom 15**：~5.4 万格 GeoJSON 约 4-6MB，fetch+渲染可控制在 2s 内且该级别用户切换不频繁；zoom 16+ 18.5 万格全量加载无收益，保持按点查询
2. **预加载 API 返回体极简化**：仅 geometry + grid_id + land_type + bbox 四个坐标字段，不返回完整属性（完整属性在框选/点选时通过 `/api/map/query` 获取），payload 缩减约 60%
3. **sticky note 用绝对定位 div 而非 Leaflet Popup**：前者可精确控制位置、动画、Z 序，后者受 Leaflet 内部约束过多（自动偏移、关闭行为固定、样式侵入性强）
4. **sticky note 弹出动画选回弹缓动**：`cubic-bezier(0.34, 1.56, 0.64, 1)` 产生轻微 overshoot，比线性 scale 更有"精致感"
